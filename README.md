# 🚀 Quick Start

📄 Clone project:
```sh
git clone https://github.com/Metapolis21/metamints-app-v2.git
```
💿 Install all dependencies:
```sh
yarn install 
```
✏ Rename `.env.example` to `.env` in the main folder and add API
